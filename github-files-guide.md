---

### 3. .gitignore

**What it is**: Tells Git which files to NOT track or upload to GitHub.

**What it excludes**:
- Operating system files (like .DS_Store on Mac)
- Editor configuration files
- Temporary files
- Log files
- Backup files
- Large media files

**Why you need it**:
- Keeps your repository clean
- Prevents uploading unnecessary files
- Reduces repository size
- Protects sensitive information
- Standard best practice

**Important**: The dot (.) at the beginning is required! Name it exactly: `.gitignore`
